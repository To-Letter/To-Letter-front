# :envelope: 온라인을 통해 만나는 아날로그한 편지, TO. Letter :mailbox_with_mail:
+ threejs를 통해 가상의 나의 방에서 편지를 주고 받을 수 있어요.
+ 내가 설정해둔 우편함 위치를 기준으로 편지가 도착하는 시간이 정해져요.
+ 나의 편지 주소를 카카오톡으로 공유할 수 있어요.
